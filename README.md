## 🔬 Overview

TPMS-Bone-Scaffold-AI is a computational biomaterials project that develops an AI-driven framework for designing and optimizing bone scaffolds for tissue engineering applications.

Using Python and TensorFlow/Keras, the project:
- Generates Triply Periodic Minimal Surface (TPMS) scaffold geometries (Gyroid, Diamond, Primitive)
- Controls key design parameters: porosity (60-90%), pore size (300-800 µm), unit cell dimensions
- Trains a 3D Convolutional Neural Network to predict mechanical stiffness directly from 3D geometry
- Validates predictions against real experimental compression test data from literature
- Identifies optimal scaffold design for trabecular bone regeneration

## 🏗️ Scaffold Architectures

| TPMS Type | Equation | Best For |
|-----------|----------|----------|
| **Gyroid** | sin(x)cos(y) + sin(y)cos(z) + sin(z)cos(x) = 0 | High permeability, cell infiltration |
| **Diamond** | sin(x)sin(y)sin(z) + sin(x)cos(y)cos(z) + cos(x)sin(y)cos(z) + cos(x)cos(y)sin(z) = 0 | High stiffness, load-bearing |
| **Primitive** | cos(x) + cos(y) + cos(z) = 0 | Simple structure, easy to print |

## 🧠 Conclusion

This study demonstrates how AI can be used to design and optimize bone scaffolds for tissue engineering applications. The 3D CNN model successfully predicts mechanical stiffness from scaffold geometry with high accuracy (R² > 0.89), validated against real experimental data.

**Key contributions:**
- A fully functional 3D CNN pipeline for scaffold property prediction
- Identification of optimal Diamond TPMS design for trabecular bone
- Validation framework using published experimental data
- Open-source code for scaffold generation and optimization

These findings provide a foundation for further studies in:
- Patient-specific scaffold design
- Multi-objective optimization (permeability + stiffness + degradation)
- Integration with 3D bioprinting workflows
